## 프로젝트 구조
회원관련 기능 , 게시판(메뉴)를 볼 수 있으며 , 분기별 기능은 다음과 같습니다.

### 메인메뉴
- 로그인
- 회원가입
- 프로그램 종료

### 회원
1. 로그인체크 
2. 회원상세 
3. 회원수정
4. 회원탈퇴

### 게시판
1. 메뉴 등록
2. 메뉴 상세읽기
3. 메뉴 수정
4. 메뉴 삭제
